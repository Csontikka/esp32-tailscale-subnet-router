MOBILE (≤639px)
───────────────

All form grids collapse to 1 column.
Label-above-input layout throughout (form-group flex-col).
Tables get horizontal scroll via .table-wrap overflow-x:auto.
Sticky form footer remains at bottom of viewport.
"+ Add" button is full-width on mobile for DHCP/port tables.

INTERACTION NOTES:
- WPA2 Enterprise fields hidden by default behind a disclosure toggle.
  Clicking "Show enterprise options" expands inline (no separate page).
- Static IP fields disabled+greyed when DHCP radio is active.
  JS toggles disabled attribute + opacity on radio change.
- DHCP "Add" opens an inline row with MAC / IP / Name inputs + Save / Cancel.
  No modal needed — inline append pattern keeps context.
- Port forward "Add" same inline-row pattern.
- Deleting a reservation/mapping: direct DELETE request, row fades out
  with CSS opacity transition then is removed from DOM.
